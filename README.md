# Library Management System

A beautiful, fast, and modern library management software built on Rust and `egui`. Custom-built for the School of Engineering and Technology to track books through offline synchronization and rapid scanning algorithms.

## Quick Start

### Running the App locally

```bash
cd library_app
cargo run
```

## Structure

- `src/main.rs`: Entry point and eframe setup.
- `src/app.rs`: State tracking and visual configuration.
- `src/ui/`: Contains the modular UI sections (`top_nav.rs`, `landing_page.rs`).
